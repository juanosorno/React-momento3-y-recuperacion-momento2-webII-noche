## Instalacion

***
Dentro del repositorio github encontramos 3 carpetas.

```Carpeta "ApiPHP"

Dentro de la carpeta "ApiPHP" entontramos la carpeta "date" la cual debemos llevar hacia nuestra carpeta xampp.

Para inicializar el proyecto php debemos abrir la carpeta "date" en nuestro IDE.
Luego ejecutamos Xampp.
```

```Carpeta "BaseDatosPhpMyAdmin"

Dentro de la carpeta "BaseDatosPhpMyAdmin" encontramos nuestro archivo de la base de datos,
para iniciarla debemos ejecutar Xampp, posteriormente abrir PhpMyAdmin y importar el archivo que contramos
dentro de la carpeta "BaseDatosPhpMyAdmin".
```

```Carpeta "React"

Dentro de la carpeta react entontramos la carpeta "dates" la cual debemos ejecutar en el IDE.

Para inicializar el proyecto react ejecutamos los comandos:
$ npm install
--para instalar las dependencias necesarios

$ npm start
--para ejecutar la aplicacion


```
